ORM - Object-Relational Mapping, объектно-реляционное отображение
ORM позволяет работать с базами данных путем запросов на синтаксисе языка пайтон,
что позволяет работать с любой базой данных, без изменения кода запросов.

MVT(MVC) (Мodel Template View) - Это паттерн проектирования позволяющий разделять отдельные части сайта,
таким образом, чтобы изменения одной части не сказывались на других

Model (модели) — отвечает за хранение и получение данных по запросу
View (представления) отвечает за то, как данные видит пользователь, или другии словами обработчик представления

Плейсхолдеры в темплейтах - {{ title }} - двойные фигурные скобки, это обращение к ключу передоваймого контекста(словаря)

Шаблонные теги - {%  %} - позволяют выполнять определенные действия, например цикл for и условие if:
{% for product in products %}
    {% if product.price > 1000 %}
        {{ product.name }} {{ product.price|default:0 }}
    {% endif %}
{% endfor %}
# тег всегда должен закрываться
# через вертикальную черту пишутся различные теги
